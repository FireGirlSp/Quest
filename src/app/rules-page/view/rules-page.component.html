Руководство

Приветствую на необычной прогулке по городу!

Как ты мог догадаться, прогулка будет с небольшим музыкальным квестом.

Короткая инструкция к сегодняшнему дню такая:

  • Будет несколько заданий, связанных с музыкой, которую ты хорошо знаешь или же точно слышал ранее
  • Точное описание каждого задания будет присутствовать на соответствующей странице приложения
  • При правильном ответе тебе откроется карта со следующим пунктом назначения, а также название места, которое тебе необходимо будет запомнить (в конце это пригодится)
  • По достижении метки, твоей второй половинке необходимо ввести код для открытия следующего задания
  • Цель прогулки - хорошо провести время и дойти до финальной точки

Удачи!
